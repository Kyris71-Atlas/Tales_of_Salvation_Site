# Tales of Salvation

Campaign archive and knowledge site for tabletop campaigns preserved with **Atlas**.

> **Status:** Early published draft — incomplete and growing.  
> Not every character dossier is finished. Soft links may 404 until discovery is complete.

## Published site

**Live:** https://kyris71-atlas.github.io/Tales_of_Salvation_Site/

Served from the repository root (`index.html`) via GitHub Pages (`main` branch).

## Local

Open `index.html` in a browser, or use any static file server from this folder.

## What this is

- **Forge of Fury** — active campaign (Tempest Company)
- **Descent into Avernus** — archived campaign (The Spice Boys)
- Character and companion dossiers as discovery completes
- Atlas doctrine under `atlas-salvation/`

## Privacy

- Fantasy Grounds campaign databases are **not** in this repo.
- `dm-private/` is ignored when present.
- Publish only what should be on the open web.
