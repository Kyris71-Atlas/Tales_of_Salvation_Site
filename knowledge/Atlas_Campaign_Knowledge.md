# Purpose

This document is the canonical knowledge representation of the campaign.

It is **not** a Fantasy Grounds export.

It is **not** a website.

It is **not** a report.

It is the trusted body of campaign knowledge produced by Atlas from
approved source systems.

Every website page, dossier, relationship map, campaign summary, and
future Atlas presentation is derived from this document and the
knowledge it contains.

------------------------------------------------------------------------

# The Workflow

``` text
Fantasy Grounds
        │
        ▼
Atlas Discovery
        │
        ▼
Atlas Objects
        │
        ▼
Atlas Campaign Knowledge
        │
        ▼
Atlas Workstation
        │
        ▼
Website
```

The normal workflow is intentionally simple:

1.  Export or copy a Fantasy Grounds campaign snapshot.
2.  Upload the snapshot to Atlas.
3.  Atlas analyzes the source.
4.  Atlas regenerates this document.
5.  Replace this one file inside `knowledge/`.
6.  Atlas Workstation rebuilds the website.

> **The user manages one file. Atlas manages everything else.**

------------------------------------------------------------------------

# Source Ownership

## Fantasy Grounds Owns

-   Campaign mechanics
-   Character sheets
-   NPC statistics
-   Combat state
-   Items
-   Spells
-   Rules implementation

## Atlas Owns

-   Discovery
-   Relationships
-   Meaning
-   Campaign history
-   Story connections
-   Session memory
-   Evidence
-   Knowledge organization

Atlas never replaces Fantasy Grounds as the source of mechanical truth.

------------------------------------------------------------------------

# Knowledge Principles

Atlas exists to transform source-system data into durable campaign
knowledge.

Every addition to this document should satisfy these principles:

-   Observe before interpreting.
-   Preserve source ownership.
-   Never guess mechanical values.
-   Record discoveries rather than assumptions.
-   Keep deterministic facts separate from narrative enrichment.
-   Prefer one canonical representation over duplicated information.
